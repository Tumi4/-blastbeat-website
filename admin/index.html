Licences & Verifiable Credentials

Issue a W3C-style licence credential per sponsored school. Each issuance is hashed (SHA-256), timestamped, and written to the audit log.

For legal / audit review

Export every issued credential plus the full audit log as one JSON bundle. The credentials follow the W3C Verifiable Credentials Data Model with a SHA-256 proof stamp. Any post-issuance change to a credential will fail proof verification.

Audit Log

Every create / edit / delete / issuance is recorded here with an ISO 8601 timestamp.
